## 题目

---

### 第一题

假设 x 和 y 的字节值分别为 0x66 和 0x39，填写下表，指明各个 C 表达式的字节值。

| 表达式 | 值 | 表达式 | 值 |
|--------|----|---------|----|
| x & y  |    | x && y  |    |
| x \| y |    | x \|\| y|    |
| ~x \| ~y|   | !x \|\| !y|  |
| x & !y |    | x && ~y |    |

### 第二题

只使用位级和逻辑运算，编写一个 C 表达式，它等价于 x==y。换句话说，当 x 和 y 相等时它将返回 1，否则就返回 0。
